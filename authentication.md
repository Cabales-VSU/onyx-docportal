# Authentication > Sign In

## 1. Mockup

## 2. Functional Description

Allows registered users to access their respective dashboards (Admin, Clerk, or Student) by providing valid credentials.

## 3. Use Case Scenario

| Step | User Action                     | System Response                                  |
| :--- | :------------------------------ | :----------------------------------------------- |
| 1    | User enters email and password. | System validates credentials against Supabase.   |
| 2    | User clicks "Login".            | System redirects to the specific user dashboard. |
